## Architecture Decision Log

1. **Groq (llama-3.1-8b-instant) over OpenAI (GPT-4o-mini)**: Originally built on GPT-4o-mini; switched to Groq free tier mid-project after OpenAI quota was exhausted. llama-3.1-8b-instant provides sufficient reasoning quality for tutoring tasks with lower latency.
2. **sentence-transformers (all-MiniLM-L6-v2) over OpenAI Embeddings**: Originally used text-embedding-3-small; switched to local CPU-based embeddings after OpenAI quota was exhausted. No API key required; embeddings quality is adequate for the course knowledge base scale.
3. **ChromaDB over Pinecone/Weaviate**: Local persistence, no external API needed, supports all required operations for the course scope.
4. **Hybrid BM25-lite + Dense over pure dense**: Keyword matching catches exact topic terms (e.g., "AVL", "O(n log n)") that embedding similarity sometimes misses.
5. **JSON-file persistence over SQLite**: Simpler setup, human-readable, sufficient for course demonstration scope. In production, SQLite or PostgreSQL would be preferred.
6. **LangGraph StateGraph over raw LangChain chains**: Required by course; also provides explicit routing logic and typed state, making debugging straightforward.
